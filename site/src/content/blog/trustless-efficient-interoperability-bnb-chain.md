<p><a href="https://www.bnbchain.org/">BNB Chain</a> is officially part of the Union network. Asset transfers and message passing for BNB Chain are now enabled on the Union network, providing fast and secure crosschain communication for BNB Chain apps and users.<br><br>This integration establishes the first trustless zk connection to BNB Chain. By leveraging zero-knowledge cryptography, Union has implemented full consensus verification for maximum security, while remaining gas efficient for crosschain messages and asset transfers.</p>
<p>With connections to Ethereum and the Babylon BTCfi ecosystem already live, and <a href="https://union.build/roadmap">more to come</a>, this integration unlocks greater access to the BNB ecosystem for users and liquidity.</p>
<h2>Custom BNB Chain Interoperability</h2>
<p>Union is the ZK interoperability L1 that’s connecting every chain across every ecosystem. The keys to Union’s interoperability are twofold: <strong>consensus verification</strong>, where validators on one chain verify that consensus has been achieved among validators on another chain, and <strong>zero-knowledge cryptography</strong>, which increase security and scalability by aggregating transfers.</p>
<p>Union’s BNB Chain integration builds on Union’s prior work by using ZK to implement consensus verification where it was previously economically unviable. BNB Chain’s large state, rapid state growth, and fast block times made the integration challenging, but the result is interoperability that is both maximally secure and gas efficient. Union interoperability is also trustless, with no reliance on centralized parties such as multisigs, oracles, or external validators.</p>
<p>For BNB Chain specifically, Union proves Parlia consensus, which is BNB Chain’s custom BFT consensus protocol that incorporates elements of Proof of Stake and Proof of Authority and is optimized for fast finality.<br><br>With this integration, apps on BNB Chain can now leverage recursive light clients and state lenses to directly communicate with any other L1 or L2 connected to Union, without needing to re-execute the transfer. That unlocks more efficient, flexible, and scalable crosschain communication for BNB applications and developers.</p>
<h2>Connecting the Next Billion Web3 Users</h2>
<p>BNB Chain’s goal is to build for the next billion Web3 users. Union is aligned with this goal—by integrating BNB Chain, Union is progressing toward its goal of connecting the next billion Web3 users, on every chain across any execution environment.</p>
